# Sales-Revenue-Analysis-Dashboard
📊 Revenue Analysis Dashboard

A modern and interactive Sales & Revenue Analysis Dashboard built with React, Recharts, PapaParse, and SheetJS. The dashboard enables users to visualize sales performance, analyze revenue trends, and gain business insights through dynamic charts, KPI cards, and advanced filtering.

✨ Features
📈 Interactive Revenue Trend Analysis
💰 KPI Cards (Total Revenue, Units Sold, Average Order Value, Top Product)
🥧 Revenue Distribution by Category (Pie Chart)
📊 Top Performing Products & Regional Sales Analysis
🔍 Product Search and Multi-level Filters
📅 Custom Date Range Selection
📄 Sortable & Paginated Transaction Table
📂 Import CSV, XLSX, and XLS files
🔄 Reset to Sample Dataset
🎨 Clean, responsive ledger-inspired UI
🛠️ Technologies Used
React.js
Recharts
PapaParse
SheetJS (xlsx)
Lucide React Icons
JavaScript (ES6+)
🚀 Key Highlights
Dynamic data visualization with real-time filtering.
Supports importing custom datasets for instant analysis.
Responsive design optimized for desktop and tablet devices.
Business-focused dashboard suitable for sales reporting and decision-making.
